# jordan-encrypt-decrypt

A small demo that provides AES-256-CBC encrypt/decrypt APIs and a static UI. Designed for deployment on Vercel.

Quick start

1. Install dependencies:

```bash
npm install
```

2. Run locally (static + API):

```bash
npm start
# then open http://localhost:3000
```

Deployment

See DEPLOYMENT.md for steps to push to GitHub and deploy to Vercel.
